# FedXCrop

Privacy-Preserving Crop Disease Detection via Federated Learning with Post-Aggregation Explainability under Non-IID Farm Distributions.

## Objective

This project extends the work of Kabala et al. (2023) by integrating:

- Federated Learning (FL)
- Non-IID farm simulation
- Grad-CAM explainability
- SHAP explainability

for crop disease detection using the PlantVillage dataset.

## Tech Stack

- Python 3.10
- PyTorch
- Flower (FL framework)
- Captum
- scikit-learn
- matplotlib

## Planned Experiments

1. Centralized baseline
2. FL IID FedAvg
3. FL Non-IID FedAvg
4. FL Non-IID FedProx
5. Grad-CAM comparison
6. SHAP explanation analysis

## Dataset

PlantVillage Dataset (Kaggle)

## Target Journal

- MDPI Plants
- MDPI Sensors
